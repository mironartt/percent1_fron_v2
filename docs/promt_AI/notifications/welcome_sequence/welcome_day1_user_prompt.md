## ЗАДАЧА

Создай ПЕРВОЕ приветственное сообщение для {{user_name}}.

### Структура сообщения:

1. **Короткое представление** (1 предложение)
   - Имя ментора не обязательно
   - Просто обозначить роль

2. **Предложение выбрать задачу** (1-2 предложения)
   - Если есть tasks_available → предложить выбрать из них
   - Если нет задач → предложить добавить одну простую
   - Фокус на ОДНОЙ задаче

3. **Призыв к действию** (1 предложение)
   - Простой вопрос или предложение

### Логика выбора задачи:

**Если есть задачи (tasks_available):**
- Выбери самую короткую по времени
- Или самую конкретную (не абстрактную)
- Предложи её как "первый шаг"

**Если нет задач:**
- Предложи добавить что-то простое
- Примеры: "одна задача на 15 минут", "что-то маленькое"

### Требования:

- Длина: 3-5 предложений
- Тон: дружелюбный, простой
- Без эмодзи
- Без мотивационных клише
- Обращение на "ты"
- Один фокус: выбрать/добавить задачу

### Примеры хорошего тона:

**Хорошо:**
"Привет, {{user_name}}. Я помогу тебе двигаться к целям. Давай начнём с малого — выбери одну задачу на сегодня."

**Плохо:**
"Добро пожаловать в OnePercent! Вместе мы достигнем невероятных высот! Ты уже сделал первый шаг к успеху!"
