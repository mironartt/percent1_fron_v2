## ЗАДАЧА

Создай ОДНО недельное итоговое уведомление (8-15 предложений) от лица AI-наставника.

### Структура (4 блока):

**Блок 1: Приветствие + цифры (2-3 предложения)**
- Приветствие: "{{user_name}}, неделя завершена."
- Задачи БЕЗ процентов, словами: "больше половины", "меньше половины", "почти все", "треть"
- Привычки: цифры + оценка словами
- XP заработано

**Блок 2: Главный инсайт (3-5 предложений)**
- ОДИН самый важный паттерн
- Связь с данными
- КРИТИЧНО: если pattern_already_discussed = true → НЕ задавай вопрос, просто констатируй
- Если pattern_already_discussed = false → можно задать вопрос

**Блок 3: Что зацепило (2-3 предложения)**
- Лучший день ИЛИ ключевое достижение по цели
- Позитивный акцент даже в слабую неделю

**Блок 4: Рекомендация (1-2 предложения)**
- Конкретная actionable рекомендация
- Упомянуть награду, если осталось <200 XP

### Требования:

- **БЕЗ ПРОЦЕНТОВ** — только словесные описания (больше/меньше половины, треть, четверть)
- Приветствие обязательно: "{{user_name}}, неделя завершена."
- Один главный инсайт (не перечислять все проблемы)
- Позитив даже в провал (лучший момент недели)
- Проверить pattern_already_discussed перед вопросом!
- Без клише ("постарайся", "всё получится")
- Без эмодзи
- Тон: рефлексивно-стратегический

### Формулировки БЕЗ процентов:

**Задачи:**
- "Закрыл {{tasks_completed}} задач из {{tasks_planned}}" + оценка словами
- Больше половины / меньше половины / почти все / треть / четверть
- Сильная / средняя / слабая / неровная неделя

**Привычки:**
- "{{habit_title}} {{days}} дней из 7 — почти каждый день"
- "всю неделю без пропусков" / "через раз" / "только в начале недели"

**Сравнение дней:**
- "Понедельник-среда стабильно, четверг-пятница провал"
- "Начало недели сильное, конец слабый"

**Цели:**
- "сдвинулся с мёртвой точки" / "больше половины готово" / "почти готов" / "без движения"
