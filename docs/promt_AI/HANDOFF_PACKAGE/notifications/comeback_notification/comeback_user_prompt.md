## ЗАДАЧА

Создай приветственное сообщение для {{user_name}}, который вернулся после паузы.

### Шаг 1: Определи категорию

**short** (3-6 дней):
- Тон: обычный
- Формат: "С возвращением" + что есть на сегодня

**medium** (7-13 дней):
- Тон: мягкий
- Формат: признание паузы + предложение начать с малого

**long** (14+ дней):
- Тон: тёплый
- Формат: приветствие + предложение определить приоритет

### Шаг 2: Сформулируй сообщение

**Структура:**

1. **Приветствие** (1 предложение)
   - Short: "С возвращением, {{user_name}}."
   - Medium: "{{user_name}}, рад что зашёл."
   - Long: "{{user_name}}, давно не виделись."

2. **Контекст** (0-1 предложение, опционально)
   - Если есть last_completed_task → можно упомянуть
   - Если нет — пропустить

3. **Предложение действия** (1-2 предложения)
   - Short: напомнить о задачах/привычках
   - Medium: предложить одну простую вещь
   - Long: предложить выбрать фокус

### Логика по категориям:

**Short (3-6 дней):**
```
С возвращением, {{user_name}}.
[Если есть задачи]: У тебя есть «{{задача}}» — начнём с неё?
[Если нет задач]: Что на сегодня?
```

**Medium (7-13 дней):**
```
{{user_name}}, хорошо что зашёл. Неделя паузы — бывает.
Давай начнём с чего-то простого. Одна задача на сегодня?
```

**Long (14+ дней):**
```
{{user_name}}, давно не виделись. Хорошо что вернулся.
Много воды утекло — давай заново посмотрим на цели. С чего хочешь начать?
```

### Требования:

- Длина: 2-4 предложения
- Тон: тёплый, без давления
- Без эмодзи
- Без "где был", "почему не заходил"
- Без статистики пропущенного
- Обращение на "ты"

### Дополнительная логика:

**Если was_on_streak = true:**
- НЕ упоминать потерянный streak (для этого есть отдельное уведомление)
- Фокус на возвращении, не на потере

**Если active_goals_count = 0:**
- Предложить создать цель или задачу
- Не говорить "у тебя ничего нет"

**Если last_completed_task существует:**
- Можно упомянуть как точку отсчёта
- "В прошлый раз закрыл «{задача}»"
