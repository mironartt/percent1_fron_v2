## ЗАДАЧА

Создай вечернее сообщение для {{user_name}} на ТРЕТИЙ день с первым Aha-моментом.

### Шаг 1: Определи тип инсайта

Проанализируй данные и выбери ОДИН тип:

**time_pattern** — если есть явный пик:
- Утром больше выполнено, чем вечером (или наоборот)
- Минимум 2-3 задачи для вывода

**task_pattern** — если есть разница по типам:
- Короткие выполняются лучше длинных
- Или наоборот

**comparison** — если есть план vs факт:
- Выполнено больше/меньше половины
- Есть с чем сравнить

**streak** — если есть серия:
- Привычка держится 2-3 дня
- Или задачи закрываются каждый день

**fallback** — если данных мало:
- Меньше 3 задач за 3 дня
- Нет явных паттернов

### Шаг 2: Сформулируй сообщение

**Структура:**

1. **Вступление** (1 предложение)
   - "{{user_name}}, прошло 3 дня."
   - Или: "Посмотрел на твои данные за 3 дня."

2. **Инсайт** (1-2 предложения)
   - Конкретные цифры
   - Наблюдение без оценки
   - Формулировка "Заметил: ..."

3. **Вопрос для рефлексии** (1 предложение)
   - "Что думаешь?"
   - "Замечал это за собой?"
   - "Это так?"

### Примеры по типам:

**time_pattern:**
"{{user_name}}, 3 дня в системе. Посмотрел на данные. Заметил: 4 из 5 закрытых задач — до обеда. Вечером пока ни одной. Утро — твой пик?"

**task_pattern:**
"{{user_name}}, прошло 3 дня. Короткие задачи летят: 4 из 4. Длинные стоят: 0 из 2. Большие задачи блокируют?"

**comparison:**
"{{user_name}}, 3 дня позади. Закрыл 5 из 8 задач — больше половины. На старте это хороший темп. Что помогало?"

**streak:**
"{{user_name}}, 3 дня. Зарядка — 3 из 3. Три дня подряд — это уже начало привычки. Как ощущения?"

**fallback:**
"{{user_name}}, 3 дня в системе. Пока данных мало для выводов — присматриваюсь к твоим паттернам. На этой неделе станет понятнее."

### Требования:

- Длина: 3-5 предложений
- Один инсайт (не перечислять всё)
- Конкретные цифры из данных
- Один вопрос в конце
- Без "молодец", "отлично", "супер"
- Без эмодзи
- Обращение на "ты"
