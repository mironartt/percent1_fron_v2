## ЗАДАЧА

Создай уведомление о потере streak для {{user_name}}.

### Шаг 1: Определи категорию

**short** (streak_was: 7-14 дней):
- Тон: лёгкий
- Акцент: один день — не провал
- Формат: факт + призыв

**medium** (streak_was: 15-30 дней):
- Тон: уважительный
- Акцент: N недель — это привычка, один пропуск не отменяет
- Формат: признание + поддержка + призыв

**long** (streak_was: 30+ дней):
- Тон: партнёрский
- Акцент: месяц+ — уже часть тебя
- Формат: констатация + уверенность + призыв

### Шаг 2: Сформулируй сообщение

**Структура:**

1. **Констатация** (1 предложение)
   - Название привычки + сколько было
   - "{{привычка}} — {{N}} дней подряд, вчера пропуск."

2. **Поддержка** (1 предложение)
   - Зависит от категории
   - Short: "Один день — не провал."
   - Medium: "{{N}} недель — это уже привычка. Один пропуск её не отменяет."
   - Long: "Месяц+ без пропусков — это уже часть тебя."

3. **Призыв** (1 предложение или кнопки)
   - "Сегодня вернёшься?"
   - Или: "[Продолжаю] [Нужна пауза]"

### Требования:

- Длина: 2-4 предложения
- Тон: спокойный, без осуждения
- Без эмодзи
- Без "к сожалению", "жаль"
- Без вопроса "почему пропустил"
- Обращение на "ты"

### Шаблоны по категориям:

**Short (7-14):**
```
{{привычка}} — {{N}} дней подряд, вчера пропуск. Один день — это не провал. Сегодня вернёшься?
[Да, продолжу] [Нужна пауза]
```

**Medium (15-30):**
```
{{привычка}} — {{N}} дней, вчера пропуск. {{X}} недель — это уже привычка. Один пропуск её не отменяет. Начать заново проще, чем кажется.
[Продолжить]
```

**Long (30+):**
```
{{привычка}} — {{N}} дней, вчера пропуск. Месяц+ без пропусков — это уже часть тебя. Один день ничего не меняет.
[Продолжаю] [Расскажи что случилось]
```

### Дополнительная логика:

**Если days_since_break > 1:**
- Адаптировать "вчера" на "{{N}} дня назад"
- Тон чуть более мягкий

**Если streak_was > previous_longest:**
- Это был личный рекорд — можно упомянуть
- "{{N}} дней — это был твой рекорд."
