# OnePercent для топ-менеджеров: как меняется продукт

---

## Новый контекст

Пользователь — не соло-предприниматель. Это топ-менеджер крупной российской компании.

У него:
- **Есть цели** — KPI, стратегические инициативы, трансформация отдела/бизнеса
- **Есть инструменты** — бюджет, команда, корпоративные ресурсы
- **Есть данные о себе** — результаты PIF (Potential in Focus, ЭКОПСИ)

Ему НЕ нужно помогать "найти цель" или "найти исполнителя на бирже фриланса". У него другая боль.

---

## Боль топ-менеджера

Топ-менеджер знает свою цель. У него есть команда и бюджет. Но между целью и результатом — он сам.

**Три ключевых разрыва:**

### 1. Разрыв "потенциал → результат"

PIF показал профиль: сильный анализ, слабая коммуникация, высокий драйв, низкая толерантность к неопределённости. Окей — и что с этим делать? Отчёт с рекомендациями лежит в столе. Потому что:
- Рекомендации абстрактные ("развивайте социальный интеллект")
- Нет конкретных шагов, привязанных к реальным рабочим задачам
- Нет системы, которая переведёт "зону роста" в ежедневные действия

### 2. Разрыв "стратегия → исполнение"

Цель понятна: "вырастить выручку направления на 40%". Но декомпозиция на конкретные шаги — это отдельная работа. А декомпозиция с учётом личного профиля (где ты силён, где слаб, что делегировать, что делать самому) — этого не делает никто.

### 3. Разрыв "знаю как → делаю"

Топ-менеджер утопает в операционке не меньше предпринимателя. Встречи, согласования, пожары. Стратегические шаги откладываются. Нет accountability-системы, которая не даст утонуть в текучке.

---

## Как меняется ядро продукта

Формула та же: **цель → декомпозиция → исполнение**. Но каждый элемент работает иначе.

### 1. Точка А: PIF вместо самодиагностики

У соло-предпринимателя точка А определяется через экспресс-диагностику. У топ-менеджера — уже есть. PIF даёт объективный профиль из 9 факторов.

Система принимает результаты PIF на входе и понимает:

| Фактор PIF | Что это значит для системы |
|---|---|
| Высокий анализ | Можно давать сложные аналитические шаги — справится сам |
| Низкая коммуникация | Шаги, связанные с влиянием на стейкхолдеров — нужна поддержка (скрипты, подготовка, тренировка) |
| Высокий драйв | Амбициозная декомпозиция уместна — не надо занижать |
| Низкая толерантность к неопределённости | Шаги должны быть конкретными, с чёткими критериями результата |
| Высокая открытость к изменениям | Можно предлагать нестандартные подходы |
| Низкая мотивация к развитию | Не грузить "развивающими" задачами — фокус на результат |

**PIF становится операционной системой декомпозиции.** Не просто "вот твои сильные и слабые стороны", а "вот как мы строим твой маршрут с учётом именно твоего профиля".

### 2. Декомпозиция: учёт профиля + корпоративного контекста

Декомпозиция строится на пересечении трёх вещей:
- **Цель** — что нужно достичь
- **Профиль PIF** — как именно этот человек эффективнее всего достигает
- **Реальные кейсы** — как аналогичные цели достигались в похожих компаниях (из открытых источников: статьи, книги, исследования, интервью руководителей)

Пример — одна и та же цель, два разных профиля:

```
Цель: "Запустить новое бизнес-направление"

Профиль А: Высокий анализ + Высокая коммуникация + Низкий драйв
→ Система: больше шагов он делает сам (анализ рынка, переговоры с партнёрами)
→ Но добавляет accountability-механизмы (дедлайны, чек-ины), потому что драйв низкий

Профиль Б: Высокий драйв + Низкий анализ + Низкая коммуникация
→ Система: аналитические шаги делает AI (исследование рынка, финмодель)
→ Коммуникационные шаги — с подготовкой (AI генерирует скрипт переговоров, ключевые аргументы)
→ Драйв высокий — амбициозный таймлайн, минимум "подстраховочных" шагов
```

### 3. Исполнение: другие исполнители

У соло-предпринимателя исполнители — AI, подрядчики, он сам.
У топ-менеджера — другой набор:

| Тип исполнения | Что делает | Пример |
|---|---|---|
| **AI делает** | Аналитика, подготовка материалов, черновики документов, скрипты переговоров | Анализ рынка, подготовка презентации для борда |
| **Делегирует команде** | Система помогает сформулировать задачу для подчинённых с учётом контекста | Поручить маркетингу исследование, финансам — модель |
| **Делает сам** | Только то, что требует его уровня: решения, переговоры, согласования | Защитить бюджет перед CEO, провести встречу с партнёром |
| **Делает сам + поддержка AI** | Шаги в зоне роста по PIF — делает сам, но с подготовкой от системы | Сложные переговоры (низкая коммуникация) → AI готовит аргументы, сценарии, возражения |

Четвёртый тип — ключевое отличие. Система не просто распределяет задачи, а **усиливает человека в его слабых зонах по PIF**.

## Пример целиком

```
Дмитрий, 38 лет, директор по развитию, производственная компания

PIF-профиль:
  Анализ:       ████████░░  8/10 (сильный)
  Изменения:    ██████░░░░  6/10 (средний)
  Коммуникация: ████░░░░░░  4/10 (зона роста)
  Драйв:       █████████░  9/10 (сильный)

Цель: "Запустить новое B2B-направление с выручкой 50M за 12 месяцев"

Шаг 1: Анализ рынка и конкурентов
  → Сам (сильный анализ — справится лучше AI)
  → AI: готовит сводку открытых данных как стартовую точку

Шаг 2: Финансовая модель направления
  → AI делает черновик → Дмитрий корректирует

Шаг 3: Презентация для борда (согласование бюджета)
  → Сам + поддержка AI (коммуникация — зона роста)
  → AI: готовит структуру презентации, ключевые аргументы,
    сценарии возражений от CFO, скрипт выступления

Шаг 4: Сформировать команду направления
  → Делегирует HR + сам (финальные интервью)
  → AI: профиль идеального кандидата из контекста цели

Шаг 5: Переговоры с первыми клиентами
  → Сам + поддержка AI (коммуникация — зона роста)
  → AI: подготовка к каждой встрече — досье на клиента,
    ценностное предложение, сценарий переговоров

Шаг 6: Построить процесс продаж
  → Делегирует команде
  → AI: помогает сформулировать задачу для руководителя продаж

Шаг 7: Запуск и первые результаты
  → Сам (дедлайн, контроль, решения)
```

| Аспект | Соло-предприниматель | Топ-менеджер |
|---|---|---|
| **Точка А** | Экспресс-диагностика | Результаты PIF (объективный профиль) |
| **Основная боль** | Вязнет в операционке, нет рук | Разрыв между потенциалом и результатом |
| **Исполнители** | AI, подрядчики, сам | AI, команда, сам, сам + поддержка AI |
| **Роль AI** | Делает работу за человека | Усиливает человека в зонах роста |
| **Подрядчики** | Фрилансеры с бирж | Своя команда и подчинённые |
| **Фокус декомпозиции** | Минимизировать работу человека | Направить работу с учётом профиля |
| **Ценность** | Экономия времени и рук | Превращение потенциала в результат |
| **Монетизация** | Токены (pay-per-use) | Скорее B2B-подписка / корпоративная лицензия |

## Добавленная ценность для топ-менеджера

### Что нового даёт система

1. **PIF → действия** — единственный инструмент, который переводит результаты оценки потенциала в конкретный план достижения конкретной цели. Сегодня PIF заканчивается отчётом. С OnePercent — отчёт становится операционной системой.

2. **Персонализированная декомпозиция** — не "как достичь цели вообще", а "как именно ТЫ достигнешь эту цель, учитывая твой анализ 8/10 и коммуникацию 4/10".

3. **AI-усиление в зонах роста** — система не просто говорит "у тебя слабая коммуникация". Она готовит тебя к каждой конкретной коммуникационной задаче: скрипт, аргументы, сценарии возражений.

4. **Развитие через результат** — человек развивает слабые стороны не через тренинги, а через реальные рабочие задачи с поддержкой. Это эффективнее любого обучения.

### Для кого это меняет правила игры

- **HR/T&D-директора** — наконец-то оценка потенциала (PIF) не заканчивается отчётом, а переходит в реальное развитие через работу
- **Сам менеджер** — получает персонального AI-операционного директора, который знает его профиль
- **Бизнес** — ROI от оценки потенциала: не "мы узнали, кто High Potential", а "мы помогли им реализовать потенциал"

## В одну фразу

**OnePercent для топ-менеджеров — система, которая берёт результаты PIF и конкретную бизнес-цель, строит персонализированный маршрут с учётом профиля потенциала, и доводит до результата — усиливая через AI там, где потенциал ещё не раскрыт.**
