## 1. Quick Add Task (Быстрое добавление задач)

### Проблема
- **КРИТИЧНО**: Невозможно добавить задачу на сегодня без перехода в Planning (∞ кликов)
- Нет кнопки "+" на Planning для создания новой задачи
- Dashboard не имеет быстрого способа добавить задачу

### Решение

**FAB (Floating Action Button)** на ключевых страницах:

```
Dashboard FAB Menu:
├── ⚡ Быстрая задача (Q)
├── 🎯 Новая цель (G)
├── 📓 Запись в дневник (J)
└── 💪 Добавить привычку (H)

Planning FAB Menu:
├── ➕ Добавить задачу (выбор дня)
└── 🎯 Создать цель
```

**Компонент**: `QuickAddTask.vue` (Modal)
- Название задачи
- Выбор дня (dropdown: сегодня/завтра/на неделе)
- Цель (выбор существующей или "Создать новую")
- Время / Приоритет (опционально)

**Keyboard shortcut**: `Cmd+Shift+A` (глобально) или `Q` (в приложении)

**Результат**: ∞ кликов → 2 клика

**Приоритет**: 🔴 P0

## 2. Breadcrumbs Navigation

### Проблема
- Непонятно где находишься в системе
- Нет контекстных переходов "назад к..."
- **Дублирование**: название страницы отображается и в Sidebar (активный пункт), и в заголовке страницы

### Решение

**Компонент**: `Breadcrumbs.vue` (reusable)

Формат:
```
Dashboard / Планирование / Неделя 13-19 января
Dashboard / Цели / [Название цели]
Dashboard / Дневник
```

### ⚠️ Важно: Убрать дублирование заголовков

**Текущая ситуация**:
```
┌─────────────────────────────────────┐
│ [Sidebar]         │  Планирование   │  ← Заголовок страницы (H1)
│  • Главная        │  ...            │
│  • ССП            │                 │
│  • Планирование ◄ │                 │  ← Активный пункт sidebar
└─────────────────────────────────────┘
```

**После оптимизации**:
```
┌─────────────────────────────────────┐
│ [Sidebar]         │ ← Dashboard / Планирование  │  ← Только breadcrumbs
│  • Главная        │  ...                        │
│  • ССП            │                             │
│  • Планирование ◄ │                             │
└─────────────────────────────────────┘
```

**Правила**:
- Breadcrumbs заменяют заголовок H1 на странице
- Последний элемент breadcrumb = текущая страница (не кликабельный)
- Убрать отдельный `<h1>Название страницы</h1>` из шаблонов всех страниц

**Файлы для изменения**:
- Создать: `src/components/Breadcrumbs.vue`
- Изменить: все page components в `src/views/` - убрать дублирующиеся заголовки

**Приоритет**: 🟡 P1
