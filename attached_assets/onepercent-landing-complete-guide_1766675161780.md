# OnePercent — Лендинг: Полное руководство с модалками

**Цель лендинга:** Получить регистрацию через демонстрацию пути пользователя в продукте

**Принцип:** 1 блок = 1 экран = 1 ключевая идея + опциональное "Подробнее"

**Дата:** 25.12.2024

---

## Общие проблемы всех блоков

### 🔴 Критические проблемы:

1. **Визуальная несогласованность**
   - **Фикс:** Все блоки на #F5F3FF

2. **Скачки по времени без объяснения**
   - **Фикс:** Переходные предложения (1-2 строки) между блоками

3. **Отсутствие единой нити повествования**
   - **Фикс:** Связующие фразы типа "Дальше произошло..."

- CTA-кнопка с конкретным обещанием
- Короткий отзыв (2 строки)

### 📂 Аккордеон "Как работает диагностика?"

**Триггер:** Текстовая ссылка или иконка (i) рядом с заголовком

**Тип:** Раскрывающийся блок внутри экрана (аккордеон)

## Блок 3: Декомпозиция (Неделя 1)

### Текущая проблема

Показывает 4 шага, но непонятно КОГДА их делать. Нет ощущения "это управляемо". Заголовок "Декомпозиция" технический.

### Что оставить как есть

- ✅ 4 конкретных шага
- ✅ Цветовая прогрессия (розовый → зелёный)
- ✅ Финальная галочка "Цель достигнута"

### Что изменить (максимум 3 пункта)

1. **Заголовок:**
   ```
   Было: "Декомпозиция"
   Стало: "Твой план на первую неделю"
   ```

2. **Добавить даты к шагам:**
   ```
   День 1 (сегодня): Выбрать вид спорта
   День 2 (завтра): Купить форму
   День 3: Первая тренировка
   День 7: Привыкнуть к ритму
   ```

3. **Акцент на поддержке:**
   ```
   Добавить: "📲 Telegram-бот будет напоминать о каждом шаге"
   ```

### Один экран должен содержать:

- Заголовок "Твой план на первую неделю"
- 4 шага с датами (визуальный центр)
- Краткое объяснение (2-3 строки)
- Упоминание Telegram-напоминаний
- Короткая цитата AI (1-2 строки)
- Отзыв (2 строки)
- Стрелка вниз (переход к следующему блоку)

### 📂 Модалка/аккордеон НЕ НУЖНЫ

**Почему:**
- План на неделю уже максимально конкретный
- 4 шага — это вся информация, которая нужна
- Добавлять больше деталей = перегружать

**Альтернатива:**
Если очень хочется дать больше контекста, можно добавить **tooltip** при наведении на каждый шаг:

```
🎯 Шаг 1: Выбрать вид спорта
   ↓ hover
   [Tooltip]
   💡 Подсказка: Выбери что ближе к дому.
   Бег, плавание или зал — главное доступность.
```

Но это опционально и не обязательно.

## Технические требования

### Модалки:

**Desktop:**
- Ширина: max 800px
- Центрирование по экрану
- Затемнение фона (overlay opacity: 0.8)
- Закрытие: по ✕, по клику вне, по ESC

**Mobile:**
- Полный экран
- Slide-in анимация снизу
- Закрытие: по ✕, по свайпу вниз

### Аккордеоны:

- Плавная анимация раскрытия (300ms)
- Иконка стрелки меняет направление
- На мобилке занимает всю ширину

### Табы:

**Desktop:**
- Горизонтальные табы сверху
- Активный таб подсвечен

**Mobile:**
- Горизонтальный скролл табов
- Свайп между контентом
