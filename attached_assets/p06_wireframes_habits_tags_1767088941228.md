### SEO-элементы страницы списка тегов привычек

| Элемент | Значение |
|---------|----------|
| Title | Теги привычек — поиск по темам • OnePercent |
| H1 | Теги привычек |
| Meta Description | 92 тега для поиска привычек по темам: утренние ритуалы, медитация, спорт, чтение, продуктивность и другие. Найдите привычку по ключевому слову. |
| Schema.org | CollectionPage, ItemList (tags), BreadcrumbList |
| Canonical | /catalog/habits/tags/ |

## Уникальные особенности тегов привычек

### 1. Временные метки
Привычки часто привязаны ко времени суток:
- `#утро` — 18 привычек
- `#вечер` — 14 привычек
- `#день` — 12 привычек
- `#перед_сном` — 8 привычек

### 2. Подборки по времени
На странице списка тегов есть специальный блок:
- "Утренние привычки" (18)
- "Вечерние привычки" (14)

### 3. Фильтр по времени выполнения
Дополнительный фильтр, которого нет у целей:
- До 5 мин
- 5-15 мин
- 15-30 мин
- 30+ мин

### 4. Связь с целями (обратная)
Страница тега привычек показывает, каким **целям** помогает этот набор привычек.
