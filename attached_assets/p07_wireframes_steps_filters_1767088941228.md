## Особенности страниц фильтров

### Общие черты с тегами

| Элемент | Теги | Фильтры |
|---------|------|---------|
| Структура | HERO → О теге → Фильтры → Список элементов | HERO → Почему фильтр → Фильтры → Список элементов |
| Пагинация | Есть | Есть |
| Связанные разделы | Есть | Есть |
| Статьи | Есть | Есть |

### Уникальные особенности фильтров

1. **Блок "Почему этот фильтр"**
   - Объясняет, кому подходит (новичкам/занятым/опытным)
   - Какие проблемы решает
   - Когда выбирать этот уровень/период

2. **Сравнительные таблицы**
   - Фильтр сложности: сравнение 3 уровней
   - Фильтр длительности: сравнение 4 периодов

3. **Навигация между фильтрами**
   - "Следующий уровень" для прогрессии
   - "Другие фильтры" для переключения измерения

4. **Кросс-фильтрация**
   - На странице сложности можно дополнительно фильтровать по длительности
   - На странице длительности — по сложности
   - Оба — по категориям

5. **Стратегии и методологии**
   - Фильтр длительности: "Стратегия 12 целей за год"
   - Фильтр сложности: "План прогрессии от лёгких к сложным"
