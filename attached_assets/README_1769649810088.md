# Trial & Subscription Notifications

Документация по уведомлениям о пробном периоде и переходе на Freemium.

## Файлы

- [USER_STATES.md](./USER_STATES.md) — Все состояния пользователя и что показывается
- [COMPONENTS.md](./COMPONENTS.md) — Описание компонентов для реализации
- [TEXTS.md](./TEXTS.md) — Все тексты для баннеров и модалок

## Краткая логика

| Состояние | UI элемент |
|-----------|------------|
| Trial (7-4 дня) | Баннер нейтральный |
| Trial (3-1 день) | Баннер warning |
| Trial (последний день) | Баннер danger |
| Trial истёк | Модалка (1 раз) |
| Freemium | Ненавязчивый баннер на Dashboard |
