| H1 | Здоровье и спорт |
| Meta Description | 50 привычек для физического и ментального здоровья: утренняя зарядка, медитация, правильное питание, качественный сон. Фильтры по времени и сложности. |
| Schema.org | CollectionPage, ItemList, BreadcrumbList |
| Canonical | /catalog/habits/health/ |
