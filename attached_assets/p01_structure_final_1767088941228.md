# SEO Каталог — Итоговая структура URL

## Краткий вывод обсуждения

**Цель:** Создать публичные SEO-страницы для органического продвижения проекта OnePercent.

**Основная идея:** Каталог целей и привычек с примерами, советами и описаниями — аналогично успешному кейсу с публичными страницами отзывов в другом проекте.

**Ключевые решения:**
1. Единый префикс `/catalog/` для всех SEO-страниц
2. Убран избыточный сегмент `/categories/` из URL
3. Шаги целей: показываются на странице цели + имеют отдельные URL
4. Теги — для перелинковки между целями и привычками
5. Bundles — готовые наборы "цель + привычки"
6. Страницы фильтрации по сложности, времени, частоте

---

## Итоговая структура URL

```
/catalog/                                              # Главная каталога
│
├── /catalog/goals/                                    # Каталог целей (главная)
│   │
│   ├── /catalog/goals/<категория>/                    # Страница категории
│   │   └── /catalog/goals/<категория>/<подкатегория>/ # Страница подкатегории
│   │       └── /catalog/goals/<категория>/<подкатегория>/<цель>/  # Страница цели
│   │
│   ├── /catalog/goals/steps/<шаг>/                    # Отдельная страница шага цели
│   │
│   ├── /catalog/goals/tags/                           # Список всех тегов целей
│   │   └── /catalog/goals/tags/<тег>/                 # Цели по конкретному тегу
│   │
│   └── /catalog/goals/filter/                         # Фильтры целей
│       ├── /catalog/goals/filter/difficulty/<уровень>/  # По сложности
│       └── /catalog/goals/filter/duration/<период>/     # По длительности
│
├── /catalog/habits/                                   # Каталог привычек (главная)
│   │
│   ├── /catalog/habits/<категория>/                   # Страница категории
│   │   └── /catalog/habits/<категория>/<подкатегория>/# Страница подкатегории
│   │       └── /catalog/habits/<категория>/<подкатегория>/<привычка>/  # Страница привычки
│   │
│   ├── /catalog/habits/tags/                          # Список всех тегов привычек
│   │   └── /catalog/habits/tags/<тег>/                # Привычки по конкретному тегу
│   │
│   └── /catalog/habits/filter/                        # Фильтры привычек
│       ├── /catalog/habits/filter/difficulty/<уровень>/  # По сложности
│       ├── /catalog/habits/filter/duration/<время>/      # По времени выполнения
│       └── /catalog/habits/filter/frequency/<частота>/   # По частоте
│
└── /catalog/bundles/                                  # Готовые наборы
    └── /catalog/bundles/<бандл>/                      # Страница набора (цель + привычки)
```

## Категории (6 основных)

Из `core/models/user_data.CATEGORY_CHOICES`:
- `welfare` (wealth) — Финансы/Благосостояние
- `hobby` (hobbies) — Хобби/Увлечения
- `environment` (friendship) — Окружение/Друзья
- `health_sport` (health) — Здоровье/Спорт
- `work` (career) — Работа/Карьера
- `family` (love) — Семья/Отношения

Подкатегории: ~30 штук (по 4-6 на категорию) — определить позже.

## Контент страниц

### Страница цели
- Breadcrumbs
- Название, иконка, категория
- Описание и преимущества
- **Шаги цели** (список на странице + ссылки на отдельные страницы шагов)
- Связанные привычки
- Похожие цели
- Теги
- CTA: "Добавить цель"

### Страница шага цели
- Breadcrumbs (категория → подкатегория → цель → шаг)
- Название шага
- Подробное описание
- Советы по выполнению
- Ссылка на родительскую цель
- CTA: "Добавить цель с этим шагом"

### Страница привычки
- Breadcrumbs
- Название, иконка, сложность, время
- Описание
- Преимущества (список)
- Советы по внедрению
- Связанные цели
- Похожие привычки
- Теги
- CTA: "Начать отслеживать"

### Страница бандла
- Название набора
- Цель (краткое описание + ссылка)
- Список привычек для достижения цели
- Общее описание стратегии
- CTA: "Добавить набор"
