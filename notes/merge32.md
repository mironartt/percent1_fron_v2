# Merge 32 — 22.12.2025

## Внесённые изменения

### 1. Блок "Рефлексия" на странице декомпозиции цели (GoalEdit.vue)

**Проблема:** Блок "Рефлексия" был уже по ширине, чем остальные блоки на странице.

**Решение:** Убраны боковые отступы у блока.

**Файл:** `src/views/GoalEdit.vue`

**Изменение CSS:**
```css
/* Было */
.goal-reflection-block {
  margin: 0.75rem 1rem 0;
  padding: 0;
}

/* Стало */
.goal-reflection-block {
  margin: 0.75rem 0 0;
  padding: 0;
}
```

**Результат:** Блок "Рефлексия" теперь по ширине совпадает с заголовком цели и другими элементами страницы.

---

### 2. Кнопка AI ментора — перемещение вправо (GoalEdit.vue)

**Проблема:** Кнопка AI ментора располагалась сразу после фильтра, а не справа.

**Решение:** Добавлен `margin-left: auto` для прижатия кнопки к правому краю flex-контейнера.

**Файл:** `src/views/GoalEdit.vue`

**Изменение CSS:**
```css
/* Добавлено */
.btn-ai-steps-inline {
  margin-left: auto;
}
```

**Результат:** Кнопка AI ментора теперь располагается справа в search-filter-bar на мобильной и десктоп версиях.
