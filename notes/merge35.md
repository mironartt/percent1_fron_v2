## 4. Замена блока "Знакомо?" на "Эффект 1%"

**Было:** Блок "Знакомо?" с 3 карточками проблем (дисциплина, доход, ступор)

**Стало:** Интерактивный блок "Эффект 1%: маленькие шаги → большие результаты"

### Функционал:
- Интерактивный слайдер (30-365 дней)
- Динамический расчёт множителя роста (формула: 1.01^days)
- Динамическое описание результата в зависимости от выбранного периода:
  - До 30 дней: "Ты начнёшь управлять днём"
  - 31-90 дней: "Система становится привычкой"
  - 91-180 дней: "Новая операционка"
  - 181+ дней: "Качественный сдвиг"

### Добавлено в script:
```javascript
const days = ref(90)
const multiplier = computed(() => Math.pow(1.01, days.value).toFixed(2))
```

### CSS-стили:
- `.effect-section` — основной контейнер
- `.slider-container` — контейнер слайдера
- `.days-slider` — стилизованный range input с градиентным thumb
- `.effect-result` — блок с результатом
- `.multiplier` — градиентный текст множителя
- `.effect-description` — динамическое описание
