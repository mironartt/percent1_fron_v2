# Merge 31: Документация для методолога

**Дата:** 17 декабря 2025  
**Тип:** Документация

---

## Описание задачи

Подготовить материалы для внешнего методолога с описанием продукта OnePercent без критически важной технической информации.

---

## Выполненные работы

### 1. Создан документ `docs/methodologist-brief.md`

**Содержание:**

| Раздел | Описание |
|--------|----------|
| О продукте | Концепция, философия 1%, форматы (web, Telegram) |
| Целевая аудитория | Текущие гипотезы, характеристики, вопросы для исследования |
| Модули продукта | Описание всех 10 модулей без технических деталей |
| Пользовательский путь | Онбординг → первая неделя → ежедневное использование |
| Геймификация | XP-экономика, streaks, достижения, награды |
| Уведомления | Типы и настройки |
| Вопросы для методолога | ЦА, retention, AI-ментор, узкие места |
| Конкуренты | Краткий обзор для сравнения |

**Модули, описанные в документе:**
1. Онбординг (5 шагов)
2. ССП / Колесо жизни (6 сфер)
3. Банк целей
4. Декомпозиция целей
5. Планирование
6. Привычки
7. Дневник (3 вопроса)
8. Достижения и XP
9. AI Ментор
10. Telegram бот

## Исключённая информация

Для безопасности и конфиденциальности **НЕ включено**:

- Техническая архитектура (Vue, Django, Vite, Pinia)
- API endpoints и структура запросов
- Внутренние метрики и KPI
- Код и структура проекта
- База данных и модели данных
- Roadmap разработки
- Бизнес-логика интеграций
